# Installation Guide

## Requirements
- Python 3.8+
- pip

## Setup
1. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file (already included) and set your `SECRET_KEY`.

4. Run the app:
   ```bash
   python app.py
   ```

## Default Pages
- `/` - Home page
- `/register` - Register page
- `/login` - Login page
- `/admin` - Admin dashboard (requires login)
- `/logout` - Logout
